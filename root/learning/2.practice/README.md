# Restaurant Data Analysis Project

This project analyzes Zomato restaurant data to understand market trends and customer preferences.

## Dataset

- `zomato.csv` - Comprehensive restaurant data including ratings, cuisine types, locations, and pricing

## Project Objectives

- Analyze restaurant distribution across different cities
- Understand cuisine preferences and popularity
- Examine pricing strategies and their correlation with ratings
- Identify factors that influence restaurant success

## Analysis Components

The project covers:
- Exploratory Data Analysis (EDA)
- Data cleaning and preprocessing
- Statistical analysis of ratings and reviews
- Geographic distribution analysis
- Cuisine popularity trends

## Files

- `main.ipynb` - Complete analysis notebook
- `zomato.csv` - Restaurant dataset

## Key Research Questions

1. What cuisines are most popular in different cities?
2. How do ratings correlate with pricing?
3. Which factors most influence restaurant success?
4. What are the geographic trends in restaurant distribution?
