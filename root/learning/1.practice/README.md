# Sales Data Analysis Project

This project focuses on exploring and analyzing sales data to derive meaningful insights.

## Dataset

The project uses the following datasets:
- `Cleaned_Sales_Data.csv` - Cleaned sales transaction data
- `Exploratory_Data_Analysis_on_Amazon's_Diwali_Campaign_Performance_2025.csv` - Amazon Diwali campaign data

## Project Goals

- Analyze sales trends and patterns
- Identify top-performing products and categories
- Understand customer behavior
- Evaluate campaign effectiveness

## Files

- `main.ipynb` - Jupyter notebook with complete analysis
- Data files in CSV format

## Key Insights

This analysis provides insights into:
- Sales performance metrics
- Seasonal trends
- Customer segmentation
- Campaign ROI analysis
